# Code of Conduct

## Our Standard

This project is intended to be a useful, open, and technically rigorous resource for the broader AI and retrieval community. Participation is expected to remain respectful, constructive, and focused on the work.

Examples of behavior that contributes to a healthy community:

- discussing ideas, designs, and failures in good faith
- giving actionable technical feedback
- respecting different experience levels and backgrounds
- documenting disagreements clearly and without personal attacks

Examples of unacceptable behavior:

- harassment, discrimination, or intimidation
- personal attacks or inflammatory language
- doxxing or sharing private information without permission
- repeated bad-faith disruption of technical discussions

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, release notes, and other project-maintained channels.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

## Reporting

For conduct concerns, contact the maintainer privately before escalating publicly where possible.
